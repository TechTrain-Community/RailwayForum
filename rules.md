# Discussions を利用する際のルールと把握しておいて欲しいこと

1. 対応してくれる Jr.Mentor は別の本業がありますので、リアルタイムでの回答が難しい場合があります。（24時間以内対応）
    - アドバイスを待っている間も、自分なりに試行錯誤しましょう！（自己解決できることがベストです）
    - メンションをつけてもらえると気づきやすいです。
2. Jr.Mentor からのアドバイスを実行し、1回で解決しない場合や次のエラーが出た場合は、再度自分で考えて、試してみましょう！（そうでないと結局 Jr.Mentor が解決することになってしまい、自分のスキルが身につきません。）もちろんそれでも解決しない場合は追加質問 ok です！
    - 悪い例）
        1. 言われた通りにやったら、次は hogehoge というエラーが出ました。とだけ伝える。
            - こうなりました。と、スクショを貼るだけも同様。
        2. Jr.Mentor のアドバイスを待っているので先に進めません。と半ば諦めてしまう。
3. Discussions の場ではテキストコミュニケーションの練習も兼ねていると思いましょう！
    - 悪い例）
        1. スクショをくださいと言われたので、ただ無言でスクショだけを貼った。
        2. 最終的にクリアできたにも関わらず、その報告をしなかった。お礼を伝えなかった。（Jr.Mentor はその後どうなったかを気にかけてくれています。）

今後、より自走力が求められたり、テキストコミュニケーションをとる機会が増えるのではないかと思います。
この Discussions を通して、そういったことの練習に少しでもなれば嬉しいです。
