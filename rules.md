# Discussions を利用する際のルールと把握しておいて欲しいこと

1. 質問への回答はボランティアで運営されています。まずは過去の質問への対応の中に、類似するものがないかを探してみましょう。
2. 試行錯誤しましょう。過去の質問の内容がそのまま適用できないこともありますが、解決の糸口になることが多いです。参考にする部分、自分で考える部分をうまく切り替えて活用してください。適宜 [TechTrain Learning Agent](https://service.techtrain.dev/learning-agent) を活用してください。
3. 敬意を持った、健全なテキストコミュニケーションをお願いします。
    - 悪い例）
        1. ただ無言でスクショだけを貼った。
        2. 最終的に解決できたにも関わらず、その報告をしなかった。お礼を伝えなかった。
