# Railway Forum

Railway Forum では TechTrain（techtrain.dev）が提供する Railway に関する質問を投稿したり、過去の質問を閲覧することができます。

質問フォーラムは [Discussions](https://github.com/TechTrain-Community/RailwayForum/discussions) よりご確認ください。

また、1分で読める [活用する際のルール](https://github.com/TechTrain-Community/RailwayForum/blob/main/rules.md) に事前に目を通していただけると、より上手に活用できると思いますのでぜひご確認ください。
